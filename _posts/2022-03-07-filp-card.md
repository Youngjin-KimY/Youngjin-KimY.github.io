## ***[march-week2] filp card***

1. how to get `gcd`, `lcm`

   - using Euclidean algorithm
      <details>
        <summary>solution</summary>

        ``` C++
        class Solution {
          public : 
            int gcd(int a, int b) {
              int t = 0;
              while(b != 0) {
                t = b;
                b = a%b;
                a = t;
              }
   
              return a
            }
   
            int lcm(int a, int b) {
              return (a * b) / gcd(a, b);
            }
        }
        ```

      </details>
